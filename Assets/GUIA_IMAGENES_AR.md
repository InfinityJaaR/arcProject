## ?? DIMENSIONES RECOMENDADAS

### Resolución mínima
- **Mínimo absoluto:** 300 x 300 px
- **Recomendado:** 512 x 512 px o más
- **Ideal para producción:** 1024 x 1024 px

### Tamaño físico del marcador
- **Mínimo para impresión:** 10 cm x 10 cm
- **Óptimo:** 15 cm x 15 cm o más
- **En pantallas:** 20 cm x 20 cm mínimo

### Aspect Ratio
- **Ideal:** 1:1 (cuadrado)
- **Aceptable:** Entre 1:0.7 y 1:1.5
- **Evitar:** Imágenes muy alargadas (< 1:0.5 o > 1:2)

## ?? TIPS PARA EDITAR TUS IMÁGENES

### En Photoshop / GIMP:

1. **Aumentar contraste:**
   ```
   Imagen > Ajustes > Brillo/Contraste
   Contraste: +30 a +50
   ```

2. **Agregar borde:**
   ```
   Editar > Trazo
   Ancho: 5-10px
   Color: Negro
   Posición: Interior
   ```

3. **Agregar texto:**
   ```
   Herramienta de Texto (T)
   Fuente: Arial Bold o similar
   Tamaño: 48pt o mayor
   Color: Alto contraste con fondo
   ```

4. **Añadir filtro de enfoque:**
   ```
   Filtro > Enfocar > Máscara de enfoque
   Cantidad: 80-120%
   Radio: 1-2px
   ```

### En Paint.NET:

1. **Contraste:**
   ```
   Ajustes > Curvas
   Forma de S pronunciada
   ```

2. **Borde:**
   ```
   Efectos > Object > Outline Object
   Ancho: 5-10px
   ```

## ?? TESTING RÁPIDO

### Antes de compilar en Unity:

1. **Test visual rápido:**
   - ¿Puedes ver fácilmente esquinas?
   - ¿Hay al menos 5 elementos distintivos?
   - ¿El contraste es claramente visible?

2. **Test de escala de grises:**
   - Convierte tu imagen a blanco y negro
   - ¿Aún se ven detalles?
   - Si todo se ve gris uniforme ? MALA IMAGEN

3. **Test de desenfoque:**
   - Desenfocar ligeramente la imagen
   - ¿Aún puedes distinguir elementos?
   - Si se vuelve una mancha ? MALA IMAGEN

## ?? COMPARACIÓN DIRECTA

| Característica | ? Mala | ? Buena |
|----------------|---------|----------|
| Contraste | Bajo (gris) | Alto (blanco/negro) |
| Detalles | < 3 elementos | > 10 elementos |
| Texto | Ninguno o poco | Múltiples líneas |
| Bordes | Suaves | Definidos |
| Esquinas | Ninguna | Muchas |
| Textura | Uniforme | Variada |
| Simetría | Total | Asimétrica |

---

## ?? CHECKLIST FINAL

Antes de usar tu imagen en ARCore, verifica:

- [ ] Resolución ? 512x512px
- [ ] Alto contraste visible
- [ ] Al menos 10 elementos distintivos
- [ ] Marcos o bordes definidos
- [ ] Texto legible (si aplica)
- [ ] Sin áreas grandes de color sólido
- [ ] No es simétrica
- [ ] No está borrosa
- [ ] Se ve bien en escala de grises
- [ ] Tamaño físico especificado en Unity
